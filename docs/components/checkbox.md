## API

### Attributes

| 属性名 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| v-model | 绑定值 | boolean / string / number | - |
| label | 选项的标签 | string / number / boolean | - |
| size | 尺寸 | 'large' \| 'default' \| 'small' | 'default' |
| disabled | 是否禁用 | boolean | false |
| border | 是否显示边框 | boolean | false |
