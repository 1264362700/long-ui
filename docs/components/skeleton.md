# Skeleton 骨架屏

在需要等待加载内容的位置提供一个占位图形。

## API 说明

### Attributes
| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| loading | 是否显示骨架屏 | `boolean` | `true` |
| animated | 是否开启闪烁动画 | `boolean` | `true` |
| rows | 骨架屏行数 | `number` | `3` |
