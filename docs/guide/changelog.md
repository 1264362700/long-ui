# 更新日志

## 1.0.0-beta.7 (2025-12-18)

### 🚀 Features
- **Apex Evolution**: 全量组件视觉质感重塑，引入三重复合投影体系。
- **TimePicker**: 新增工业级时间选择器组件。
- **Homepage V2**: 官网首页极致视觉升级，支持 3D 转换与噪声玻璃态。

### 🐞 Bug Fixes
- 修复 `Switch` 组件在生产构建时的 SCSS 变量冲突。
- 修复 `InputNumber` 计数器在特定模式下的 BEM 语法错误。
- 修复 `Preview.vue` 在 VSCode/VitePress 下的 TS 类型推断问题。
