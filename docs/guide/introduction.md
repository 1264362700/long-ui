# 介绍

Long UI 是一个现代化的 Vue 3 组件库，采用 Monorepo 架构，提供高质量的 UI 组件。

## 特性

- 🚀 **Vue 3** - 基于 Vue 3 Composition API
- 📦 **TypeScript** - 完整的类型定义
- 🎨 **主题定制** - CSS Variables 驱动的主题系统
- 🌲 **Tree-shaking** - 按需加载，减少打包体积
- 📝 **文档完善** - VitePress 驱动的组件文档

## 设计原则

- **一致性** - 统一的视觉和交互设计
- **高效性** - 简洁的 API，开箱即用
- **可定制** - 灵活的主题系统
- **可访问性** - 符合 ARIA 标准
