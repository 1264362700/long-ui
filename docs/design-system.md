# long-ui UI 设计规范 (Ant Design 级)

> 本文档用于确保后续所有组件开发的视觉一致性。

## 1. 核心颜色 (Colors)
- **品牌色 (Primary)**: `#1677ff` (AntD Blue)
- **成功色 (Success)**: `#52c41a`
- **警告色 (Warning)**: `#faad14`
- **错误色 (Error)**: `#ff4d4f`
- **标题色**: `rgba(0, 0, 0, 0.88)`
- **正文色**: `rgba(0, 0, 0, 0.65)`
- **辅助色/失效色**: `rgba(0, 0, 0, 0.25)`
- **边框色**: `#d9d9d9` (默认), `#f0f0f0` (Table/Divider)

## 2. 圆角与边框 (Borders & Radius)
- **常规圆角**: `6px`
- **输入框/按钮**: `6px`
- **边框宽度**: `1px`

## 3. 阴影 (Shadows)
- **聚焦阴影 (Focus Ring)**: `0 0 0 2px rgba(5, 145, 255, 0.1)`
- **悬浮遮罩 (Popovers/Modals)**: `0 6px 16px 0 rgba(0, 0, 0, 0.08), 0 3px 6px -4px rgba(0, 0, 0, 0.12), 0 9px 28px 8px rgba(0, 0, 0, 0.05)`

## 4. 尺寸 (Dimensions)
- **Default (32px)**: 基础高度，Input/Button/Select 默认。
- **Large (40px)**: 强引导或大场景。
- **Small (24px)**: 表格内嵌或紧凑布局。

## 5. 动画过渡 (Transitions)
- **时间**: `0.2s`
- **曲线**: `cubic-bezier(0.645, 0.045, 0.355, 1)`

## 6. Table 规范
- **Header 背景**: `#fafafa`
- **单元格内边距**: `12px 16px` (Default), `8px 8px` (Middle/Small)
- **行 Hover 背景**: `#fafafa`
